# Winit 최애와 소통하는 가장 빠른 방법
여러분의 최애는 누구인가요? <br>
저의 최애는 아이돌도, 스포츠 선수도 아닌
저의 곁에 있는 가족, 연인, 친구라고 하고 싶습니다. <br>
당신의 최애와 소통하세요!
